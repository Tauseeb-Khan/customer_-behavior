\#Customer Shopping Behavior Analysis 



\#Project Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and MySQL. The goal is to extract meaningful business insights from raw retail data and understand purchasing patterns.



\#Dataset Information

The dataset contains customer-level shopping details, including:



\- Demographics (Age, Gender)

\- Product Information (Category, Item Purchased)

\- Purchase Data (Purchase Amount, Frequency)

\- Customer Behavior (Previous Purchases, Subscription Status)

\- Transaction Attributes (Payment Method, Shipping Type, Discount Applied)



\#Tools \& Technologies Used

\- Python (Pandas, NumPy)

\- SQL

\- MySQL

\- Jupyter Notebook

\- Git \& GitHub



\#Key Data Processing Steps

✔ Data Cleaning \& Preprocessing  

✔ Missing Value Handling  

✔ Feature Engineering  

✔ Age Group Segmentation  

✔ Purchase Frequency Transformation  

✔ Redundancy Removal  

✔ Database Integration  



\#Key Insights Extracted

\- Customer spending patterns across categories

\- Age group purchasing behavior

\- Impact of discounts on purchase amount

\- Purchase frequency analysis

\- Customer retention indicators



\#Business Value

This analysis helps businesses:



\- Understand customer segments

\- Optimize pricing \& discount strategies

\- Improve marketing targeting

\- Enhance customer experience
